Use your GoPro, like a PRO


Magic Lantern GoPro (MLGP) is a tool to improve the imaging and lens capabilities of the little action camera. MLGP provides hacks from Lock Exposure to Video effects, including Fisheye removal and much more.
At the moment, it only supports the GoPro HERO3 Black / + Black and + Silver.
If you have a Hero3 White or Hero3 Silver you can contact me to test MLGP.

How does this work?

MLGP is a library of files called autoexec.ash , these files act as a autoexecutable for GoPro cameras, the .ash extension is for Ambarella Shell. To install a hack onto a gopro camera, just Right click the GitHub RAW button and choose save as, save the file in the root level of the SD card. Much more info about how to install the hack on the gopro and a troubleshooting here.
The core of MLGP is the command "t ia2", that command in Ambarella Shell hacks the lens.

MLGP is FREE and Open Source! This means that all hacks are completely free and with MIT license, you can contribute sending PRs or issues.
