GoPro Long Exposure pictures/timelapse hack

Get those awesome long exposure timelapses or pictures with your GoPro HERO3 Black or Plus Black using this hack.
To download it, change the parameters and save it to your gopro go to the github repo hack here The code looks like this t app appmode photo sleep 1 t ia2 -ae still_exp P X Y sleep 1 t app button shutter PR sleep 2 d:\autoexec.ash REBOOT yes

Key:
P: ISO
X: Exposure time
Y: Iris (exposure time also?)
