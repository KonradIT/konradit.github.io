GoPro camera research

2013 - Present

I have been developing scripts for GoPro cameras and researching their WiFi API calls. I first got a GoPro in 2012, loved it and decided to extend the use by writing an shell script to extend the timelapse interval. Some of the highlights:
  • Successfully extend the timelapse interval on HERO2/HERO3 cameras, which was used by several companies and hobbyists.
  • Made the Ambarella Shell script that enabled the HERO3 Black camera to shoot long exposures.
  • Currently maintain the unofficial GoPro API (goprowifihack), a great resource for hobbyists and companies.
