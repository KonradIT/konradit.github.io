HERO BUS HACKS

All GoPro HERO cameras have a bus to connect "BacPacs", accesories that improve the camera by attaching them onto the back of the camera.

But that bus is a new way to use the camera. To make new BacPacs! (RC Wired/Wireless remotes, Sensorial Triggering, etc...)

BUY CONNECTORS

Cam-Do Connectors

ChargeConverter Connectors

LIST OF PINS

THE CONNECTORS HAVE 30 PINS

HERE ARE THE FUNCTIONS OF EACH PIN:

Examples:

Pin 30 + Pin 1 = PWR/MDE Button, Turn on the camera by connecting those pins for 2 secs, To start recording / take pictures / record a timelapse NOTE: ONE BUTTON MODE Must be ON.

Pin 2: Composite/Red RGB. Composite is output when Pin 20+21 are tied to the GND

Pin 5: External 5 volt power supply. Apply the external power to this pin. Weâ€™re not dependent on internal camera power supply, but can run for hours with the external power.

Pin 23: Connect this to the ground using a LED and a 150-ohm resistor, and the LED will light up when the camera is turned on.

WiFi Commands, WiFi Live Preview, WiFi Server

WiFi Commands and Live Preview code on my GitHub repo

The GoPro HERO cameras have a extraordinary WiFi server, you can browse media, live preview what your GoPro sees, or control all the settings of your camera.

LIVE PREVIEW

1. Turn the WiFi of your camera in GOPRO APP mode
2. In your computer, connect to the GoPro's WiFi
3. Go here in your browser: http://10.5.5.9:8080
4. Go to Live
5. You should see a file called "amba.m3u8" (If not, wait and refresh browser)
6. Open VLC media player
7. File > Open network
8. Write this in the text field: http://10.5.5.9:8080/live/amba.m3u8
9. Now, you have the preview.

EMBEDDED PREVIEW

You can embed a preview for Windows using VLC WEBplugin:

<embed type="application/x-vlc-plugin" name="video1" autoplay="no" loop="yes" width="400" height="300" target="http://10.5.5.9:8080/live/amba.m3u8" /> <br /> <a href="javascript:;" onclick='document.video1.play()'>Play video1</a> <a href="javascript:;" onclick='document.video1.pause()'>Pause video1</a> <a href="javascript:;" onclick='document.video1.stop()'>Stop video1</a> <a href="javascript:;" onclick='document.video1.fullscreen()'>Fullscreen</a>

Alternatively, you can embed the preview using JW Player.

write this in your head tag:

<script src="http://jwpsrv.com/library/9Xd9rv1IEeKDbSIACusDuQ.js"></script>
Customizable JW Player:

<div id='playernPcydqsAvglZ'></div> <script type='text/javascript'> jwplayer('playernPcydqsAvglZ').setup({ file: 'http://10.5.5.9:8080/live/amba.m3u8', title: 'GoPro HERO Live Stream', width: '100%', aspectratio: '16:9', fallback: 'false', autostart: 'true' }); </script>

WIFI COMMANDS

URL structure:

http://10.5.5.9/camera or bacpac/COMMAND?t=PASSWORD&p=%NUMBER

PASSWORD: Replace by your camera's WiFi Password

Example: http://10.5.5.9/bacpac/SH?t=PASSWORD&p=%01

I have made a GitHub Repo with loads of information about the wifi commands, live preview, media browsing, get the current state... GoProWiFiHack repo
