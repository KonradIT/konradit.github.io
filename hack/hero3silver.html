HERO3 Silver

The HERO3 Silver has the same processor as the HERO2, so the HERO2 commands work with the HERO3 Silver

Key
t app key: up/up_rel/down/down_rel/left/left_rel/right/right_rel/zoomin/zoomin_rel/

zoomout/zoomout_rel/set/focus/focus_rel/shutter/shutter_rel/record/

mode/menu/power/del/format/copydir/switchosd

 Example: t app key zoomin
Shutter button
t gpio 43 sw out0 / 1 t gpio 43 sw out0
Note:Start Recording: t gpio 43 sw out0

Stop Recording: t gpio 43 sw out1
t gpio 43 sw out0
Take Picture: t gpio 43 sw out0

Turn Off
t gpio 48 sw out0
sleep 3
t gpio 43 sw out0		 Example: t gpio 48 sw out0
sleep 3
t gpio 43 sw out0
Front-Back LEDs
t gpio 37 sw out0/1, 0 = ON, 1 = OFF Example: t gpio 37 sw out0
sleep 8
t gpio 37 sw out1
Top-Bottom LEDs
t gpio 46 sw out0/1, 1 = ON, 0 = OFF Example: t gpio 37 sw out1
sleep 8
t gpio 37 sw out0

If you find any command that does not work, contact me.
