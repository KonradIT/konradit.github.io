HERO

The HERO is the basic entry level GoPro camera

Modes
t app modes photo/video Example: t app modes photo
Usb config
t app usb mtp/msc (mtp=GoPro's encrypted USB connection. msc=Mass storage connection) Example: t app usb msc
Delete a directory (such as DCIM) (works with all cameras!)
deletedir: SD card port d:\ + DCIM folder Example: deletedir d:\DCIM
Shutter button
t app msh msh Example: t app msh msh
Video resolutions
t app videores 1080p30 (1080 30fps)
t app videores 720p60 (720 60fps)
t app videores 720p60sv (720 60fps SuperView)		 Example: t app videores 720p60sv
Lock exposure
t ia2 -exp lock 1 Example: t ia2 -exp lock
