HERO4 Black

The HERO4 Black has the new Ambarella A9 processor, and new os.

LCD Bacpac settings
t drv bacpac on/off Example: t drv bacpac off/td>
Print the temperature
t drv temps Example: t drv temps
Delete a directory (such as DCIM) (works with all cameras!)
deletedir: SD card port d:\ + DCIM folder Example: deletedir d:\DCIM
10 Pin USB female connector settings
t drv show/enable/disable Example: t drv show
LED settings
t drv led
start top/bottom/front/back blue/red miliseconds ON miliseconds OFF
stop top/bottom/front/back blue/red
enable top/bottom/front/back blue/red
disable top/bottom/front/back> blue/red		 Example: t drv start top blue 600 0
Log the temperature to the sd card
t drv templog on/off Example: t drv templog on
Shutter button
t appc capture start/stop/still (start=start video, stop=stop video, still=take picture) Example: t appc capture still
Wifi streaming
t appc stream start/stop Example: t appc stream start
LiveView settings
t appc liveview start/stop Example: t appc liveview start
MultiShot capture
t appc multishot_capture start/stop Example: t appc multishot_capture start
show bluetooth pairing code
t appc wireless pin Example: t appc wireless pin
Protune settings
imgcap
video/still sharpness low/medium/high
video/still color_mode gopro/flat
video/still wb auto/3000k/5500k/6500k/off
video/still ev_comp -2.0 - 2.0		 Example: imgcap wb auto
